## 2. Locked decisions (operator-confirmed, not revisitable without explicit override)

- **Polaxory is an art studio with the agent as principal artist.** Murakami / Kaikai Kiki frame.
- **Jetski (Blake) is the named operator.** Not anonymous. The artist's CV is the project's CV.
- **Goal:** bigger-than-self, capital engine responsible to the art, built to outlast the crypto cycle.
- **Token venue:** Clanker on Base. 40% creator / 60% protocol fee split confirmed.
- **RobloxOps is Shape A** — a Polaxory surface, not a separate business. Wave-making, not wave-catching.
- **Launch sequence:** build-first / token-as-go-live, not same-day stack. Phase 1 private build on existing credits, Phase 2 public ignition when go-live criteria met.
- **Inference architecture:** layered and vendor-agnostic. Claude for high-stakes work, cheap open-source (Groq / Mistral / DeepSeek / Gemini Flash) for routine. Self-hosting deferred.
- **Tools:** open-source preferred — BloxBot, RbxLuauLLM, ClawBlox. PromptBlox explicitly demoted: paid SaaS is death when we could build/own.
- **No-waste principle:** every internal capability is a potential revenue stream, filtered by the brand-amplifies-vs-strips test.
- **Boundaries:** never church data, never AI output as human-generated, treasury keys env-vars-only, never off-character.

## 5. Research findings worth carrying

- **Clanker on Base:** Farcaster cast-based launch. 1B fixed supply. 40% creator / 60% protocol on the 1% Uniswap V3 trading fee. ~13K launches/day, $8M record protocol fee week in Feb 2026. Multi-chain now.
- **x402 is production-ready** as of mid-2026. 69K active agents, 165M txns, $50M cumulative volume. SDKs in TS/Go/Python. Foundation members include Anthropic, AWS, Google, Visa, Circle, Vercel. AWS Bedrock added native support May 7, 2026.
- **Open-source Roblox AI generation exists:** BloxBot (MIT), RbxLuauLLM, ClawBlox, RMod, LM Games. No need for PromptBlox subscription.
- **Inference economics (2026):** Groq Llama 8B ~$0.06/M tokens, Gemini Flash $0.10/$0.40, Claude Sonnet ~$3/$15. Self-hosting break-even ~256M tokens/month. Launch volume is ~2-5M tokens/month total. Inference is not the dominant cost of running Polaxory.
- **Roblox 2026 TOS (April 30 update):** AI-generated content explicitly allowed. Disclosure required. User responsible for outputs.
- **Current Roblox dominant narratives (NOT Backrooms):** avatar customization (green avatar trend, Roblox face trend), social hangouts as community spaces, simulators with steady progression. Identity-expression > horror IP right now.
- **Roblox Open Cloud API:** lets you POST `.rbxlx` files to existing places. Cannot create new universes from scratch — placeholder must be pre-created manually on Creator Hub. OAuth flow available (PromptBlox uses `universe.place:write`).
- **PromptBlox specifics:** $9.99/mo base, generates obbies/tycoons/paintball/simulators in ~60 sec, no fleet management, no analytics. Competitors: Lemonade.gg, SuperbulletAI.
