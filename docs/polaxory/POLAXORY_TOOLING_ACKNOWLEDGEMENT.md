# Polaxory Tooling Acknowledgement

Polaxory has a quiet tooling layer running behind the repo.

It exists to preserve continuity across builder, validator, research, and release-readiness loops. It should not become the headline of the project page.

## What the acknowledgement should say

```text
Tooling Layer: active
Quiet unless shipping, blocked, or validating a release claim.
```

## What it should not do

- Compete with the product thesis.
- Add agent spectacle to the GitHub page.
- Expose private workspace state.
- Present automation as a guarantee of correctness.

## Why this exists

The tooling is there so Polaxory can keep moving after context shifts: one artifact, one commit, one next step.
