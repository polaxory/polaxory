# ADR 0002: Shared vs Local Data

## status
Accepted

## decision
Universal identity, cosmetics, social/party state, settings, and safety state are platform-owned. Progression, currencies, power items, quests, rankings, balance state, and saves are local to each sector.

## consequences
- no global currency.
- platform storage remains a registry rather than a universal mechanic bucket.
- sectors control their own migrations and balance changes.
- portable data stays intentionally small and legible.
